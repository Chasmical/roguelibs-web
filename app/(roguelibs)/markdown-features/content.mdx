## Code blocks

`````mdx title="/markdown-features/code-blocks.mdx"
You can highlight your code blocks by appending the language code to the triple backticks.
We use **PrismJS** for syntax highlighting - it [supports almost 300 different languages](https://prismjs.com/#supported-languages)!

```cs
public class MyAwesomeItem : CustomItem
{
    [RLSetup]
    public static void Setup()
    {
        RogueLibs.CreateCustomItem<MyAwesomeItem>();
    }
}
```
`````

:::info{title="Syntax language aliases"}
RogueLibs also defines a few useful language aliases to make things easier:

- You can write `cs` instead of `csharp`;
- `xml`/`html`/`svg`/`csproj` instead of `markup`;
- `js`/`ts` instead of `javascript`/`typescript`;
- `md`/`mdx` instead of `markdown`;
- `sh`/`shell` instead of `bash`;
- `ps` instead of `powershell`;
- *and some more*.
:::

:::info{title="Customization options"}
- `title=""` allows you to specify the *"path"* to the file;
- `nocopy` removes the Copy button;
- `nonums` removes line numbers;
- `wrap` makes the code wrap, by default. The wrapping is not pretty at the moment, but it will be worked on!
:::
